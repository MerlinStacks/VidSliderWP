## üì• Installation

1.  Upload `reel-it` to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Navigate to **Reel It** in the admin menu.

## üõ†Ô∏è Usage

### Creating a Gallery
1.  Go to **Reel It > Galleries**.
2.  Click **Create Gallery** (or edit the default one).
3.  Click **Manage Videos** to add videos from your Media Library or upload new ones.

### Tagging Products
1.  Ensure **WooCommerce** is active.
2.  In the "Manage Videos" modal, click **Tag Products** next to a video.
3.  Search for a product and select it.
4.  A product card overlay will now appear on that video.

### Displaying the Slider
**Option A: Block Editor**
*   Search for the "VidSliderWP" block and add it to your post.
*   Select your feed from the Block Settings.

**Option B: Shortcode**
*   Copy the shortcode from the "Galleries" page:
    ```
    [reel_it feed_id="YOUR_ID" use_feed="true"]
    ```
*   Paste it into any shortcode block or text widget.
